## The stack used:
* Java 20;
* Gradle 8.3;
* Javalin
* H2, PostgreSQL;
* Lombok;
* JUnit

## Requirements:
Before you can build this project, you must install and configure the following dependencies:
* JDK 20
* Gradle 8.3
* Node.js 16.13.1
* Makefile

## Start application:
Build application
> make build

Run application on localhost (default port - 7070)
> make run-dist
